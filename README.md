## 프로젝트 소개
- 기존 가계부의 직접 작성해야 하는 불편함과 MZ 세대의 양면적인 소비 행태를 반영하여 소비 주체로서의 MZ 세대를 만들기 위해 기획
- 간편성을 추구하기 위해 UI 간편화, 통계 기능 간편화, 계좌/카드 내역 API 연동을 도입
- 짧은 호흡을 중시하고 SNS를 즐기는 MZ 세대의 특성을 반영하기 위해 챌린지를 도입
- 양극적 소비 특성을 반영하여 한달 목표 금액을 도입
